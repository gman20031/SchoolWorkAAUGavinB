level creation tutorial:
Case matters for everything
need a @ to be the player, only 1
need a x to be the exit
include a | or - to create enemies
include captial O to create traps
periods ( . ) are empty spaces
all maps must be rectangles and the same width every line
Spaces between characters are NOT required, but all levels will be displayed with characters between.
Nothing will yell at you if you fail to do this, but it will not work properly.
